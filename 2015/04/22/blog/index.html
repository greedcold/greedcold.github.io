Middleman весьма простой в использовании генератор статичных сайтов. Он также хорошо подходит для создания блога. Опишу один из способов создание и деплоя блога на Github pages.

Установка

Открываем свой терминал и вводим 

$ gem install middleman

Далее создаем репозиторий на Гитхабе, важно чтобы имя репозитория было формата <имя-аккаунта>.github.io.

Далее заходим в папку где вы хотите создать свой проект и инициализируете блог, связывая его с репозиторием гитхаба 

middleman init username.github.io --template=blog

Аргумент --template=blog используется для создания временной платформы структуры блога

Вводим в терминал команду для запуска сервера и видим заготовку для блога на localhost:4567 

$ middleman server

Настройки

Добавляем в наш проект Livereload,который позволяет вносить изменения в свой код, а затем (если ваш сервер работает) смотреть изменения в браузере без необходимости обновлять страницу.Вписываем в gemfile

gem "middleman-livereload", "~> 3.3.0"

Далее нужно активировать Livereload в config.rb файле

activate :livereload

Для создания нового поста впишите в терминал

$ middleman article название_статьи

Middleman deploy

с помощью гема middleman-deploy можно легко задеплоить статичную страницу на Github pages.

Для начала добавим гем в gemfile 

gem "middleman-deploy", "~> 1.0"

Далее открываем файл Config.rb и добавляем конфигурацию middleman-deploy сразу после конфигураций блога

activate :deploy do |deploy|
          deploy.method = :git
          deploy.branch = 'master'
          deploy.build_before = true
        end
         
        activate :directory_indexes

Далее вводим:

middleman deploy

Переходим на username.github.io в браузере и видим свой бложик, осталось только настроить внешний вид.

Примечание

При добавлении поста нужно каждый раз делать деплой.
